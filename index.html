Swimmaker, a website to design Swimming Workouts

By Gabriel Navarro

Start designing your Swimming Workout below this line

First, choose what today's set will focus on:


Now you can start designing the Warm-Up

First, choose how many rounds the Warm-Up will be.

Now, choose which stroke the Warm-Up will be.



Now, you can start designing the Pre-Set.

First, choose how many rounds the Pre-Set will be.

Now you can start choosing what goes into the Pre-Set.
Start by saying which stroke you want it to be (e.g. 50 Free), and add more text fields in the order you want them to be swam. (If it is a drill, say something like "50 Back 6-3-6 Drill").



Now, you can finally start designing the Main Set

Start by choosing how many rounds the Main Set is going to be.

Now choose which strokes the Main Set will be, the same way you chose the Pre Set strokes.



Lastly, design a Warm Down

Start by choosing how many rounds the Warm Down will be.

Finally, choose which strokes the Warm Down will be.



Now that you've designed the entire Workout, it will all show up down here, copyable into a Google Doc for ease of printing [WIP]
